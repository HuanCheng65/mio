This project (Mio) is a Koishi plugin that simulates a personified QQ group member, driven by LLMs.

This directory is the plugin directory, the entire Bot project directory is two levels up.

## Rules you MUST follow all the time
- Always address me as "Echo", at the beginning of every response.
- DO NOT write any code for backward compatibility unless I explicitly ask for it.
- Use yarn for package management.

## How to build

Enter the entire Bot project directory and run `yarn build`.
